## 배터리·발열·소음 실사용 체크


고성능 노트북은 스펙 숫자만 보면 좋아 보이지만, 실제로 쓸 때 발열과 소음이 작업을 방해하면 의미가 없습니다.


**1. 배터리 — 80.2Wh의 실사용 시간**


80.2Wh는 16형 노트북 중 대용량 구간입니다. 웹 브라우징·문서 작업 같은 가벼운 사용에서는 약 8~10시간, 영상 편집·렌더링 같은 고부하 작업에서는 약 2~3시간으로 체감이 크게 달라집니다. RTX 5070 GPU가 동작하면 전력 소모가 급증하므로, 고성능 작업 시에는 항상 충전기를 준비해야 합니다.


**2. 발열 — 고부하 시 표면 온도 체크**


Core Ultra 7 + RTX 5070 조합은 풀로드 시 키보드 상단부와 힌지 부근에서 열감이 올라옵니다. 삼성의 쿨링 시스템(듀얼 팬 + 히트파이프)이 내부 온도를 관리하지만, 장시간 렌더링이나 게임에서는 팜레스트까지 미온이 전달될 수 있습니다. 일반 모드에서는 발열이 거의 느껴지지 않습니다.


**3. 소음 — 성능 모드별 차이**


일반 사무 작업 시에는 팬이 거의 돌지 않아 무소음에 가깝습니다. 성능 모드를 올리거나 GPU 부하가 걸리면 팬 속도가 올라가며 약 40~45dB 수준의 소음이 발생합니다. 카페나 도서관에서 고부하 작업을 하면 주변에 들릴 수 있는 수준입니다.
