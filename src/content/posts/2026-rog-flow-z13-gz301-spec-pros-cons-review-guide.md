## ROG Flow Z13 GZ301, 게임 성능 체감은 어떨까?


13형 태블릿 폼팩터에서 게임이 된다는 게 핵심 매력입니다. 하지만 "어느 정도"까지 되는지가 구매 결정의 핵심입니다.


**AAA 게임 FHD 기준** -- RTX 4050/4060 외장 GPU 탑재 SKU 기준으로, FHD 해상도에서 사이버펑크 2077 중간 설정 40~55fps, 발로란트 고설정 100fps 이상을 기대할 수 있습니다. 울트라 설정에서 60fps 고정은 어렵지만, "이동하면서 게임"이라는 폼팩터를 고려하면 충분한 수준입니다.


**XG Mobile 외장 GPU 독 연결 시** -- GZ301은 ASUS XG Mobile 포트를 지원합니다. 외장 GPU 독을 연결하면 데스크톱급 성능에 근접할 수 있지만, 독 가격이 본체에 버금가므로 비용 대비 효과를 따져야 합니다.


**발열·소음 현실** -- 13형 고성능 설계라 게임 시 팬 소음이 확실히 큽니다. 터보 모드 기준 45~50dB 수준으로, 헤드셋 착용이 사실상 필수입니다. 키보드 상단 온도도 40~45도까지 올라가므로 쿨링 패드를 권장합니다.
