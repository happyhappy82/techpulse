# 스펙·체감·주의점, 구매 전 반드시 체크할 3가지


**1. 스펙 체감 — CPU 성능은 넉넉하지만 GPU에 기대하지 마세요**


Core Ultra 5 225H는 14코어 구성으로 문서 작업, 브라우저 다중 탭, 화상회의를 동시에 처리하기에 충분합니다. AI Boost NPU가 탑재되어 향후 AI 기반 앱 최적화에서도 이점이 있습니다. 하지만 내장 그래픽 중심이라 영상 편집 렌더링이나 게임에서는 확연한 한계가 있습니다. 그래픽 작업 비중이 높다면 비보북 16X나 외장 GPU 모델을 검토해야 합니다.


**2. 체감 차이 — 메모리 확장 가능성이 장기 만족도를 좌우합니다**


X1607CA의 장점 중 하나는 메모리 슬롯 1개가 있어 추후 RAM 확장이 가능하다는 점입니다. 기본 16GB로 시작하더라도 필요 시 추가 장착으로 32GB까지 올릴 수 있어, 온보드 고정 메모리인 경쟁 모델 대비 장기 사용 유연성이 높습니다. 다만 16형 WUXGA 60Hz 패널은 고주사율 게이밍보다는 문서·학습 중심 사용에 더 적합합니다.


**3. 주의점 — 모델코드 혼동과 OS 포함 여부**


비보북 16 라인은 X1607CA 외에도 유사한 코드(M1605YA 등)가 함께 검색됩니다. 같은 "비보북 16"이라도 CPU 세대, RAM 타입, SSD 용량이 완전히 다를 수 있으므로, 반드시 'X1607CA' 코드를 결제창까지 확인해야 합니다. OS 포함 여부(프리도스 vs Windows 11)도 가격 차이의 주요 원인이니 간과하지 마세요.
