## 실구매 전 꼭 확인할 3가지


### 1. RTX 4060 TGP(전력 한도) 확인

- **같은 RTX 4060이라도 TGP에 따라 성능이 20~30% 차이**납니다.
- TUF Gaming A14의 RTX 4060은 SKU에 따라 TGP가 75W~100W까지 다릅니다.
- TGP가 높을수록 게임 FPS가 높지만, 발열과 팬 소음도 비례합니다.
- **확인 방법**: 판매 페이지 상세 스펙에서 "GPU TGP" 또는 "최대 그래픽 전력"을 찾으세요. 표기가 없으면 판매자에게 직접 문의하는 것을 권합니다.

### 2. SKU별 CPU·RAM·패널 차이 점검

- "TUF Gaming A14 4060"이라는 이름 아래 실제로 여러 SKU가 존재합니다.

| 확인 항목 | SKU별 차이 예시                           | 영향                       |
| ----- | ------------------------------------ | ------------------------ |
| CPU   | AMD Ryzen 9 8945HS vs Ryzen 7 8845HS | 멀티코어 성능 10~15% 차이        |
| RAM   | 16GB DDR5 vs 32GB DDR5               | 게임+스트리밍 병행 시 32GB 유리     |
| 패널    | 165Hz FHD vs 165Hz QHD               | QHD에서 4060 부담 증가, FHD 권장 |
| SSD   | 512GB vs 1TB                         | AAA 게임 3개 이상이면 1TB 필요    |

- **핵심**: 모델명만 보지 말고, CPU 모델넘버 + RAM 용량 + 패널 해상도를 결제 직전에 확인하세요.

### 3. 발열 설계와 소음 허용치 확인

- 14형 컴팩트 바디에 RTX 4060을 넣었기 때문에 쿨링 설계가 성능을 좌우합니다.
- **터보 모드**: 최대 성능이지만 팬 소음이 45~50dB까지 올라갑니다. 헤드셋 필수입니다.
- **밸런스 모드**: 소음은 35~40dB로 줄지만 GPU 클럭이 낮아져 FPS가 10~15% 감소합니다.
- **저소음 모드**: 팬 소음을 최소화하지만 게임 성능이 크게 제한됩니다.
- **키보드 표면 온도**: 고부하 30분 이상 시 WASD 영역 약 40~45도. 장시간 게임 시 손바닥이 따뜻해집니다.
- **확인 방법**: 구매 전 유튜브에서 동일 SKU의 벤치마크·발열 테스트 영상을 찾아보는 것을 강력 권합니다.
