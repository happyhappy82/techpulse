## RAM·SSD 업그레이드, 가능한가?


노트북을 저렴하게 사고 나중에 RAM이나 SSD를 직접 늘리는 전략은 가성비를 극대화하는 방법입니다. NT750XHD-KC51G에서 이 전략이 통하는지 확인합니다.


### RAM 업그레이드

- NT750XHD 시리즈는 **온보드(납땜) + SODIMM 슬롯 혼합 구성**인 경우가 많습니다.
- 온보드 8GB + 빈 SODIMM 슬롯 1개 구성이라면, 8GB 또는 16GB 모듈을 추가로 장착해 총 16~24GB로 확장할 수 있습니다.
- **주의**: 온보드 전량 납땜(LPDDR5) 모델은 RAM 교체/추가가 불가합니다. 구매 전 해당 SKU의 메모리 타입(LPDDR vs DDR5 SODIMM)을 반드시 확인하세요.
- **추천 전략**: 기본 8GB 모델을 사고 나중에 16GB SODIMM을 추가하면, 처음부터 16GB 모델을 사는 것보다 3~5만 원 절약할 수 있습니다.

### SSD 업그레이드

- NT750XHD-KC51G는 **M.2 NVMe SSD 슬롯**을 사용합니다.
- 기본 256GB 모델을 구매한 경우, 기존 SSD를 512GB~1TB로 교체하거나, 빈 슬롯이 있다면 추가 장착이 가능합니다.
- 삼성 980 Pro, WD SN770 등 NVMe SSD 가격이 1TB 기준 8~12만 원 수준이므로, 직접 교체 시 비용 효율이 좋습니다.
- **주의**: SSD 교체 시 기존 윈도우를 클론하거나 재설치해야 합니다. 클론 소프트웨어(삼성 Data Migration 등)를 미리 준비하세요.

### 업그레이드 난이도

- 하판 나사 제거 후 커버를 열면 RAM 슬롯과 SSD 슬롯에 접근할 수 있습니다.
- 유튜브에 "NT750XHD 분해"로 검색하면 영상 가이드를 참고할 수 있습니다.
- 난이도는 중하 수준으로, 정전기 방지 장갑과 플라스틱 오프너만 있으면 초보자도 가능합니다.
- **삼성 AS 정책**: 사용자 직접 업그레이드 시 공식 보증이 제한될 수 있으므로, AS 정책을 미리 확인하세요.
