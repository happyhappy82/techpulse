## RAM·SSD 업그레이드 가이드


### RAM: 온보드 방식 — 구매 시 결정이 최종


KD72S는 LPDDR5X 온보드 메모리를 사용합니다. 메인보드에 직접 납땜되어 있어 **구매 후 RAM 추가·교체가 물리적으로 불가능**합니다.


**어떤 용량을 선택해야 하나?**

- **16GB**: 문서 편집, 웹 브라우징, 화상회의 중심이라면 충분합니다. 브라우저 탭 30개 이내, 문서 앱 2~3개 동시 사용 수준.
- **32GB**: 코딩(IDE + 터미널 + 브라우저), 영상 편집, 가상머신, 대용량 엑셀 작업이 있다면 32GB를 강력 추천합니다. 한번 정하면 바꿀 수 없으므로 넉넉하게 가는 것이 안전합니다.

### SSD: 교체 가능 — 향후 확장 여지 있음


KD72S는 M.2 NVMe SSD 슬롯을 사용하며, 기본 1TB 구성입니다. 향후 용량이 부족해지면 2TB·4TB SSD로 교체할 수 있습니다.


**SSD 교체 시 주의사항**

1. 하판 나사 제거 후 내부 접근 (삼성 서비스센터 또는 자가 교체)
2. 기존 데이터 백업 필수 (외장 SSD 또는 클라우드)
3. PCIe 4.0 NVMe 규격 호환 SSD 사용
4. OS 재설치 또는 클론 작업 필요
