## 직장인 업무용 최적 세팅 가이드


### 업무 환경별 활용법


**일반 사무직 (문서·이메일·회의)**


Core Ultra 7 + 16GB RAM이면 엑셀 대용량 시트, PPT 100페이지 이상, Outlook + Teams + 브라우저 탭 20개 이상을 동시에 띄워도 여유가 있습니다. 16형 16:10 화면은 세로 비율이 넓어 문서 작업 시 스크롤이 줄어들고, 화면 분할로 양쪽에 자료를 띄우기에 최적입니다.


**기획·마케팅·데이터 분석**


1TB SSD는 보고서 파일, 데이터셋, 프레젠테이션 자료를 넉넉하게 보관할 수 있습니다. 피벗테이블 대량 처리, BI 도구 연동, 간단한 데이터 시각화 작업도 Core Ultra 7이면 무난하게 소화합니다.


**개발·IT 직군**


IDE + Docker + 브라우저 동시 실행 환경에서 16GB RAM은 최소 사양입니다. 개발 작업 비중이 높다면 **32GB 옵션**을 선택하는 것이 안전합니다. 1TB SSD는 프로젝트 파일과 개발 환경을 여유 있게 관리할 수 있습니다.
